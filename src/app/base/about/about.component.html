Scheduler

Is an hobby app which is used to help you manage Your day. It features Activities, Tasks and Shopping Lists management. With two first you can track your weekly challenges easily and ensure yourself, You won't forget about anything. The very useful things are also of course Shopping Lists, which You can create very easy, add items, mark them bought. You can also share Your Tasks and Shopping Lists with others. Just play with it for some time.

This project is mainly created for better CV presentation and of course for gain skills at web application programming. It is written using Angular and Spring Boot which both I started to love some time ago.
